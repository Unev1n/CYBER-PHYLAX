
# 🛡️ Cyber Phylax

**Train Before You Become a Target.**

Cyber Phylax is an interactive cybersecurity awareness platform that teaches users how to identify and respond to common cyber threats through realistic simulations instead of passive reading.

## 📖 About

Cyber Phylax is designed to improve cybersecurity awareness by placing users inside real-world attack scenarios.

Instead of reading lengthy guides, users actively make security decisions and instantly learn the consequences of their actions.

The project aims to build practical cybersecurity skills for everyone—from students to everyday internet users.

## 🎮 Current Simulations

### 1. Email Phishing

Learn how to identify:

- Suspicious sender addresses
- Fake links
- Urgency tactics
- Generic greetings
- Payment scams

Users can choose to:

- Click Link
- Report Email
- Ignore Email

Each decision leads to a different outcome and educational explanation.

---

### 2. Fake Website Detection

Users inspect a realistic login page and learn to verify:

- Website URL
- HTTPS
- Domain names
- Design inconsistencies
- Credential theft attempts

---

### 3. Government Website Awareness

Users learn to identify:

- Official government domains
- Fake portals
- Scam payment requests
- Phishing messages pretending to be government agencies

## 🎯 Purpose

Cyber Phylax focuses on one simple idea:

> People learn cybersecurity better by experiencing attacks than by reading about them.
